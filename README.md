# TesisWeb

Sitio web estático que publica y presenta la tesis profesional de Ingeniería
Mecatrónica **«Reconocimiento y clasificación de objetos usando inteligencia
artificial basada en SVM y visión estereoscópica»** (Elvis Franks Condori Arias,
UNI, 2013).

🌐 En línea: **https://elviscondori.github.io/**

## Contenido

- Portada, resumen, problema, objetivos, capítulos, resultados y aplicaciones.
- Presentación de la sustentación (Prezi embebido).
- Juego interactivo «Sé el hiperplano» para entender el SVM.
- Descarga del PDF completo de la tesis y cómo citarla.

## Stack

HTML, CSS y JavaScript plano, sin frameworks ni build tools. Publicado en
GitHub Pages.
